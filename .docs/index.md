# MyProfile Suite

プロフィール帳アプリとログインボーナス管理アプリを備えた、やさしい自己表現＆日課サポートのミニアプリ集です。

## What We Provide

感情設計を重視したUIで、好きなことの共有や日々のモチベーションづくりを支援します。現在はプロフィール帳アプリが利用可能で、ログインボーナスアプリを順次追加します。

## Products

### MyProfileNote App
カテゴリ選択・入力・プレビュー・PNG保存までを `/profile` 配下で提供するプロフィール帳アプリ

### Login Bonus App (Coming Soon)
日替わり・週替わりのボーナス配布を柔らかい演出で設定できるアプリを `/login-bonus` 配下に追加予定

## Target Users

- 学生ユーザー: 友人とプロフィールカードを交換したい人
- 趣味コミュニティ: 安心感のある自己紹介カードを用意したい参加者
- ゲーム／コミュニティ運営者: ログイン継続を促したい管理者（予定）

## Core Value Proposition

複数のやさしい体験を1つのプロジェクトに束ね、パステルトーンのUIと即時プレビューで「好き」と継続習慣をサポートします。

## Context (Optional)

Cloudflare Workers 上の Hono API との連携を視野に入れ、テンプレート共有やログインボーナス配信 API を今後統合する計画です。
