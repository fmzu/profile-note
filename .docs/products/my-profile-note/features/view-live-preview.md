# View Live Preview

入力フォームと同期したプレビューカードをリアルタイムで表示する機能です。

## Steps

1. `/profile/create` ページでフォームにフォーカスが当たると、対応するプレビュー要素がハイライトされる。
2. ユーザーが文字を入力するたびに Reducer 経由でストアが更新される。
3. プレビューカードは最新の状態を参照し、遅延なく描画を更新する。
4. プレビューの背景と装飾は選択したテーマに従い、PNG 化しやすい構造を保つ。

## Acceptance Criteria

- 入力とプレビュー表示にズレが生じない。
- 文字数制限を超えた場合はプレビュー側に ellipsis を適用し、可読性を保つ。
- プレビュー領域は `html-to-image` で PNG 化できる DOM 構造を維持する。
