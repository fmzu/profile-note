# Create Profile Card

カテゴリ選択と入力フォームを通じて、プロフィールカードを完成させる機能です。

## Steps

1. `/profile/select` ページでプリセットまたは自由入力からカテゴリを最大4件選択する。
2. `/profile/create` ページに遷移し、なまえ／ひとこと／各カテゴリの内容を入力する。
3. 入力内容がストアに保持されていることを確認しながら、プレビューカードを更新する。
4. 必須項目が揃ったら「できた！」アクションを実行し、`/profile/done` へ遷移する。

## Acceptance Criteria

- 4件を超えるカテゴリは選択できない。
- 入力内容は Zustand ストアに保持され、ページ間遷移でも失われない。
- 不足入力がある場合はやさしいメッセージで完了ボタンを非活性にする。
