# Route /done

完成ページは、画像保存のガイドと再作成導線を提供し、達成感を演出するページです。

## Purpose

作成完了を祝福しながら、ユーザーがカードをPNGとして保存できるようにします。

## Layout

- 完了メッセージとフェードアニメーション
- 保存手順（「保存ボタン」→ ダウンロード）と注意事項
- 再作成ボタン、別テーマ選択への導線（拡張余地）

## Navigation

- 「もう一度つくる」で `/select` に戻る（状態初期化）
- 保存ガイド内のボタンで `html-to-image` 処理を実行
