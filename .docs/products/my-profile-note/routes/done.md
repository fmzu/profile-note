# Route /profile/done

完成ページは、PNG 保存のガイドと再作成導線を提供し、達成感を演出するページです。

## Purpose

カード完成を祝福し、ユーザーが安心して PNG として保存できるようにします。

## Layout

- 完了メッセージとやさしいトーンの案内
- 保存手順（ボタン押下で html-to-image 経由のダウンロード）
- 再作成ボタンと「もう一度カテゴリをえらぶ」導線

## Navigation

- 「PNGで保存する」でダウンロード処理を実行
- 「もういちどつくる」で `/profile/select` に戻る（リセット後）
