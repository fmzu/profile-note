# Route /profile/create

入力＋プレビュー画面は、プロフィールカードの内容を編集しながらリアルタイムで完成形を確認できる中心ページです。

## Purpose

ユーザーが安心して内容を入力し、プレビューで仕上がりを即確認できるようにします。

## Layout

- 左カラム: なまえ／ひとこと／カテゴリ別テキストエリアの入力フォーム
- 右カラム: 背景カラーを反映したプレビューカード（スマホ縦長想定）
- 下部: 保存アクションボタンと Tips 表示

## Navigation

- 戻るボタンで `/profile/select` に戻る（状態は保持）
- 「できた！」ボタンで `/profile/done` へ遷移（必須入力が揃った時のみ活性）
