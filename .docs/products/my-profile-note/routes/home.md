# Route /profile

プロフィール帳トップページは、アプリの概要とフロー開始の導線を提供します。

## Purpose

ユーザーに体験の価値を伝え、「つくってみる」アクションでカテゴリ選択へ誘導します。

## Layout

- ヒーローメッセージとアプリ紹介テキスト
- パステルトーンの背景と角丸カード
- メインCTAボタン（`/profile/select` へ遷移）

## Navigation

- プライマリアクションで `/profile/select` に進む
- ルートトップ（`/`）への戻り導線を最小限に留め、今後のアプリ追加時に再利用
