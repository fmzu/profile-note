# Route /profile/select

カテゴリ選択ページはプリセットと自由追加を組み合わせ、最大4件まで「すき」を選べるインターフェースを提供します。

## Purpose

ユーザーが表現したいカテゴリを選び、次の入力ステップの土台を整えます。

## Layout

- プリセットカテゴリカードのグリッド表示
- 自由追加フォームと追加ボタン
- 選択済みカテゴリ一覧と残り選択数カウンター

## Navigation

- 戻るボタンで入口ページ（`/`）に戻る
- 「つぎへすすむ」ボタンで `/profile/create` へ進む（1件以上選択時のみ活性）
